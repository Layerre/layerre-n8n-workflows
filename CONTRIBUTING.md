# Contributing

## What to submit

- Practical templates that solve a real use case (e.g. social posts, certificates, bulk images).
- No hardcoded API keys, no personal data. Use sticky notes for overview and steps; use markdown with `## H2` headings. Give nodes clear names.

## Requirements

1. **Sticky notes:** One yellow overview sticky with setup and description; white/neutral stickies for each main step.
2. **Formatting:** Markdown only, `## H2` headings (no HTML).
3. **Nodes:** Descriptive names (e.g. "Create Template from Canva", not "HTTP Request").
4. **Data:** Sample or clear instructions; no real credentials or PII.

## How to submit

1. Export your workflow as JSON from n8n (kebab-case filename, e.g. `my-workflow-name.json`).
2. Add it to the `templates/` folder and add a short entry in README under "Templates".
3. Open a pull request.

Follow [n8n template guidelines](https://n8n.notion.site/Template-submission-guidelines-9959894476734da3b402c90b124b1f77).
